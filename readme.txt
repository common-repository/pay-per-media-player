== Installation ==

1. Copy folder "pay-per-media-player" into your wordpress plugins folder, normally located in /wp-content/plugins/

2. Login to Wordpress Admin and activate the plugin


A HTML5 Video playlist player can be embedded in a post using a tag of the following form:

[payper id=ID]  i.e. [payper id=1]

 
Please make sure your playlist ID after customizing XML with "payper player settings" form.


You can also set global options in the payper media player option menu.
